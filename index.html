Questions and Answers

Question-1: How does JavaScript work?

Answer: The browser where we run JavaScript code has javascript engine. When we run JavaScript code in our browser, the engine receives the code. Then it parses the code line by line. Then it turns the code into machine code. After that, it executes the code. This is how JavaScript works.

Question-2: What is the difference between Local Storage and Session Storage?

Answer: The main difference between Local Storage and Session Storage is the data stored in session storage is temporary and the data stored in local storage is not temporary. It means that session storage data exists as long as the browser is open. But local storage data exists even after closing the browser. After closing the browser, the session storage data gets deleted. But local storage data still remains even after closing the browser.
